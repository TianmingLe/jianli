## 1. 产品概述

胡亚伟个人作品集网站 —— 一名视觉设计师 / AI 设计师 / 品牌设计师的暗色系作品集。以"克制中的科技感"为核心美学，通过视频背景、大字排版、留白与微动效，塑造高级、冷峻、有未来感的个人品牌形象。

- 目标用户：潜在客户、品牌方、设计同行、招聘方
- 核心价值：以最克制的方式传递最强烈的个人能力印象，区别于模板化作品集
- 设计基调：暗色系 / 高级 / 克制 / 科技感 / 非模板

## 2. 核心功能

### 2.1 用户角色
| 角色 | 访问方式 | 核心权限 |
|------|----------|----------|
| 访客 | 直接访问 | 浏览全部内容、点击联系链接跳转 |

### 2.2 功能模块（单页滚动 + 锚点导航）
1. **Hero 首页**：全屏视频背景、大标题、导航栏、联系按钮

4. **个人优势模块**：卡片展示核心能力
5. **底部联系模块**：整屏收尾页，联系方式汇总

## 4. 用户界面设计

### 4.1 设计风格
- **主色**：近黑底 `#0A0A0B` / `#0D0E10`，配合 `#111214` 卡片层
- **强调色**：冷冽电光蓝 `#7DD3FC` 或金属银白 `#E5E7EB`，克制使用
- **文字**：主文 `#F5F5F7`，次文 `#A1A1AA`，弱化 `#52525B`
- **按钮**：极简描边 + hover 填充，无圆角或极小圆角（2-4px）
- **字体**：标题用 distinctive display 字体（如 Syne / Space Grotesk 替代品），正文用 refined sans（如 Manrope / Sora），中文用思源黑体 / 霞鹜文楷
- **布局**：版心 1700px，大量留白，网格但有意打破
- **氛围**：噪点纹理、渐变光晕、细线分割、微动效（fade-up / stagger）
- **图标**：lucide-react，线性极细风格

### 4.2 页面设计概览
| 模块 | 视觉要点 |
|------|----------|
| Hero | 全屏视频 + 渐变遮罩 + 超大标题（10vw+）+ 顶部毛玻璃导航 + 右下角滚动提示 |
| 个人经历 | 左侧人物图（去背/单色处理）+ 右侧文字 + 底部数据条 |
| 精选项目 | 大卡片纵向堆叠，每张近全宽，hover 时图片缩放 + 信息上浮 |
| 个人优势 | 4-6 张能力卡片网格，hover 描边亮起 |
| 底部联系 | 整屏大留白 + 居中超大 "Let's Create" + 联系方式列表 |

### 4.3 响应式
- 桌面优先（PC 端展示），版心 1700px
- 基础响应式适配（≤1024px 单列、≤768px 简化布局），但本次以 PC 体验为主

### 4.4 视频背景指引
- 使用免版权暗调氛围视频（如 Pexels / Coverr 的科技/抽象类素材）
- 叠加 `linear-gradient` 暗色遮罩保证文字可读性
- `autoplay muted loop playsinline`，移动端降级为静态图
